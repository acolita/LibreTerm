# LibreTerm v1.1.0 Plan (Tab Management & View Modes)

**Goal:** Improve user productivity by providing advanced tab management options and maximizing screen real estate with new view modes.

## 1. Advanced Tab Management
*   **Feature:** Extend the Tab Context Menu (`IDR_TAB_MENU`) with more closing options.
*   **New Menu Items:**
    *   **"Close Others":** Closes all tabs except the currently selected one.
    *   **"Close to Right":** Closes all tabs to the right of the selected one.
    *   **"Rename Tab":** Allows the user to temporarily rename the active tab (useful for tracking specific tasks).

## 2. Fullscreen & Focus Modes
*   **Fullscreen (`F11`):**
    *   Toggles the application into a borderless fullscreen mode.
    *   Hides: Title Bar, Menu Bar, Sidebar (TreeView), Status Bar.
    *   Restores to previous state on toggle.
*   **Focus Mode (`Ctrl+B` or View Menu):**
    *   Toggles the Sidebar (Connections Tree) visibility to give more width to the terminal without going full fullscreen.

## 3. Status Bar Enhancements
*   **Feature:** Display active session details in the Status Bar.
*   **Data:** Show `User@Host (Port)` and `Connection Name` when a tab is selected.
*   **Implementation:** Update `OnTabChange` to refresh Status Bar text.

## 4. Technical Implementation Steps
1.  **Resources:**
    *   Add Menu Items: `IDM_TAB_CLOSE_OTHERS`, `IDM_TAB_CLOSE_RIGHT`, `IDM_TAB_RENAME`.
    *   Add View Menu: `IDM_VIEW_FULLSCREEN`, `IDM_VIEW_SIDEBAR`.
    *   Add Accelerators: `F11` for Fullscreen.
2.  **UI Logic (`MainWindow.cpp`):**
    *   Implement `ToggleFullscreen()`: Use `GetWindowLong`/`SetWindowLong` to change style and `ShowWindow` for controls.
    *   Implement `ToggleSidebar()`: Show/Hide `m_hTreeView` and `m_hSearchEdit` and adjust layout in `OnSize`.
    *   Implement Tab Logic: Loop through tabs and send `WM_CLOSE` (or internal close logic) based on index.
    *   Implement `RenameTab`: Simple `DialogBox` with an edit field, then `TabCtrl_SetItem`.
3.  **Status Bar:**
    *   In `OnTabChange`, extract `Session` data and `SendMessage(m_hStatusBar, SB_SETTEXT, ...)`

## 5. Tests
*   **Manual/E2E:**
    *   Open 3 tabs. "Close Others" -> Only 1 remains.
    *   Open 3 tabs. Select middle. "Close Right" -> 1 closes.
    *   Press F11 -> App covers screen, no chrome. Press F11 -> Restores.
