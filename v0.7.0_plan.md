# LibreTerm v0.7.0 Plan (Power User Features)

**Goal:** Add high-productivity features for power users, specifically Multi-Input (Broadcasting) and enhanced UI interactions.

## 1. Multi-Input (Broadcast Mode)
*   **Feature:** A toggleable mode where typing in one terminal sends the same keystrokes to all other *visible* or *active* terminals.
*   **UI:** Add a "Multi-Input" button to the Toolbar or a menu item in "Tools".
*   **Implementation:**
    *   Intercept keystrokes in the `MainWindow` (using the same hotkey logic or a message hook).
    *   Loop through all active `Session` objects and `PostMessage` the `WM_CHAR` or `WM_KEYDOWN` to their `hEmbedded` handles.

## 2. Advanced Navigation (TreeView)
*   **Drag & Drop:** Support dragging connections into different folders.
*   **Sorting:** Automatically sort connections and folders alphabetically.
*   **Inline Rename:** Press `F2` on a TreeView item to rename it directly.

## 3. Tab Convenience
*   **Mouse Shortcuts:** 
    *   Middle-click on a tab to close it.
    *   Double-click on a tab to close it.
*   **Reordering:** Allow dragging tabs to reorder them (using `TCM_SETITEM` and manual logic).

## 4. Quick Connect
*   **Feature:** A "Quick Connect" toolbar button or shortcut (`Ctrl+Q`).
*   **UI:** A simple dialog with just Host, User, Port, and Pass that doesn't save to the permanent list (unless the user clicks "Save").

## 5. Technical Implementation Steps
1.  **Tab Mouse Hooks:** Subclass the Tab Control to handle `WM_MBUTTONUP` and `WM_LBUTTONDBLCLK`.
2.  **Multi-Input logic:** Create a `BroadcastInput(UINT msg, WPARAM wParam, LPARAM lParam)` method.
3.  **Sorting:** Implement a `TVSORTCB` callback for `TreeView_SortChildren`.
4.  **Drag & Drop:** Use `TVN_BEGINDRAG` and `ImageList_BeginDrag` for visual feedback.
