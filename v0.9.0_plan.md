# LibreTerm v0.9.0 Plan (Organization & Convenience)

**Goal:** Empower users to organize their connections efficiently and connect faster with Quick Connect and Import/Export features.

## 1. Quick Connect
*   **Feature:** A menu item to launch a connection immediately without saving it to the permanent list.
*   **UI:** A simplified dialog (Host, User, Port, Pass).
*   **Shortcut:** `Ctrl+Q`.

## 2. Sorting
*   **Feature:** Automatically sort folders and connections alphabetically in the Sidebar.
*   **Implementation:** Use `TreeView_SortChildren` or logic during `ReloadConnections`.

## 3. Data Portability (Import/Export)
*   **Export:** Save all connections to a portable `connections.json` file.
*   **Import:** Load connections from a `json` file (merging with existing).
*   **Logic:** Implement a basic JSON generator/parser in `ConnectionManager`.

## 4. Drag & Drop Organization
*   **Feature:** Move connections between folders in the Sidebar using Drag & Drop.
*   **Implementation:** Handle `TVN_BEGINDRAG`, `MouseMove` tracking, and `WM_LBUTTONUP` re-parenting.

## 5. Technical Implementation Steps
1.  **Quick Connect:** Add Resource IDs, Dialog template, and Menu item.
2.  **Sorting:** Update `ReloadConnections` to ensure alphabetical insertion or call sort.
3.  **Import/Export:** Extend `ConnectionManager` with JSON support.
4.  **Drag & Drop:** Implement the Win32 TreeView drag-and-drop boilerplate.
